# SCRIB.block

**Block Me If You Dare**

A static, installable, offline-ready single-page PWA for beginner-friendly drawing prompts. Built for Procreate users, sketchbook gremlins, and anyone trying to defeat the blank page without turning art into homework.

## Files

- `index.html` — complete app, styles, and JavaScript
- `manifest.webmanifest` — installable PWA metadata
- `service-worker.js` — offline cache
- `icons/ICON-192x192.png` and `icons/ICON-512x512.png` — official SCRIB.block app icons
- `icons/apple-touch-icon.png` — iOS home screen icon

## Deploy

Upload the folder to GitHub Pages, Netlify, Cloudflare Pages, Vercel, or any static host. No build tools. No APIs. No backend.

## Notes

The app saves Stash items in `localStorage`, so saved prompt relics stay on the device/browser used.


## Update: Expanded Choice Banks

This build preserves the original preselected prompt choices and adds expanded banks for subject types, objects, foods, blobs, lettering, shape language, twist types, twist ideas, lesson focus, Procreate layer setups, reflection questions, and prompt title patterns.

The app is still static, single-page, offline-ready, and deployable on free static hosts.
